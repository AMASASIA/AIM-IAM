# AIM3 Notebook Timeline Extension

## 🎯 概要

AIM3のPersonal Notebookに**Timeline/Calendar可視化機能**を追加しました。
Markwhen風のシンプルなテキスト記法で、美しいタイムラインとカレンダーを生成できます。

## ✨ 機能

- **テキストベースのタイムライン作成**: ````timeline` ブロック内にイベントを記述
- **Timeline/Calendar切替**: ボタン一つで表示形式を切り替え
- **Glassmorphismデザイン**: AIM3の既存UIと完全に統合
- **タグサポート**: `#tag` でイベントを色分け
- **日付範囲対応**: 単一日付と期間の両方をサポート

## 📝 使い方

Personal Notebookで新しいエントリーを作成し、以下のように記述します：

\`\`\`markdown
# プロジェクトロードマップ

\`\`\`timeline
2026-02-01: コンセプトデザイン
2026-02-10 - 2026-02-20: プロトタイプ開発 #coding
2026-02-14: バレンタインリリース ❤️
2026-03-01: グローバルローンチ 🚀
2026-03-15 - 2026-04-15: ユーザーフィードバック期間 #testing
Now: 現在の開発スプリント
\`\`\`

プロジェクトは順調に進んでいます！
\`\`\`

## 🎨 タグカラー

以下のタグは自動的に色分けされます：

- `#coding` - 紫
- `#design` - ピンク
- `#meeting` - ティール
- `#launch` - エメラルド
- `#milestone` - アンバー
- `#release` - 赤
- その他 - ブルー（デフォルト）

## 🔧 技術仕様

### 新規追加ファイル

1. **TimelineBlock.vue** - タイムライン/カレンダー表示コンポーネント
   - Timeline View: 横スクロール可能なガントチャート
   - Calendar View: 月表示のカレンダーグリッド
   - イベントリスト: 詳細情報付きリスト表示

2. **MarkdownRenderer.vue** (拡張)
   - ````timeline` ブロックの検出と解析
   - 既存のMarkdown機能を維持
   - Vue render関数による動的コンポーネント統合

### 記法仕様

```
YYYY-MM-DD: イベント名 #tag          // 単一日付イベント
YYYY-MM-DD - YYYY-MM-DD: イベント名   // 期間イベント
Now: 現在のマイルストーン            // 現在位置マーカー
```
